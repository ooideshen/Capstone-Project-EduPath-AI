# edupath_ai_backend
Getting Started
1) Install JDK version 24 (jdk-24_windows-x64_bin.exe)
2) Unzip edupath_ai_backend file from GitHub
3) Open file in IDE (recommended IntelliJ IDEA) 
4) Install pgAdmin4 (https://www.pgadmin.org/download/) and run pgAdmin4.exe file
5) Create a database known as edupith_ai_db in pgAdmin4
6) Change the application.properties file into your pgAdmin password.
7) Run AiApplication.java
8) Run data.sql file to load the data
